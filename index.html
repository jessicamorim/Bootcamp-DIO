Tag - p

A tag p é para parágrafos. Exemplo:

O elemento HTML representa um parágrafo. Os parágrafos geralmente são representados em mídia visual como blocos de texto separados de blocos adjacentes por linhas em branco e/ou recuo na primeira linha, mas os parágrafos HTML podem ser qualquer agrupamento estrutural de conteúdo relacionado, como imagens ou campos de formulário.
